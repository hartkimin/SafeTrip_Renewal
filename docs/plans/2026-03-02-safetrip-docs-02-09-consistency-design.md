## 각 문서 처리 프로세스

### Step 1 — Explore 에이전트 (불일치 탐색)
- 대상 문서 전체 읽기
- 상위 기준 문서들의 관련 섹션 읽기
- **불일치 항목 목록** 출력 (각 항목: 위치 + 현재 내용 + 기준 내용 + 변경 이유)

### Step 2 — Fix 에이전트 (수정)
- Explore 결과 기반으로 **해당 위치만** 수정
- 전체 구조·문체·원본 내용 유지 (불일치 부분 외 NO TOUCH)
- 수정 후 변경 이력 섹션에 항목 추가

### Step 3 — Review 에이전트 (검증)
- "수정해야 할 것만 수정했는가?" 확인
- "상위 문서와 이제 일치하는가?" 확인
- Pass → 다음 문서 / Fail → Fix 재의뢰

> **작성일**: 2026-03-02
> **목적**: Master + #02~#34 (34개 문서)를 비즈니스 원칙 v5.1 기준으로 점검하고 불일치 부분만 수정

## Goal

작성지침 3개를 제외한 **Master + #02~#34 (총 34개 문서)**를 `01_T1_SafeTrip_비즈니스_원칙_v5.1.md` 기준으로 점검하고 불일치 부분만 수정한다.
**원칙**: 상위 문서(비즈니스 원칙 v5.1)가 절대 기준 → 하위 문서를 맞춤. 원본 내용 최대한 보존.

## Architecture — 2-Phase

### Phase 1: Explore (전체 탐색, 6개 그룹 병렬)

6개 Explore 에이전트를 동시에 투입해 불일치 항목을 수집한다.

| 그룹 | 대상 문서 |
|------|---------|
| G1 | Master_SafeTrip_마스터_원칙_거버넌스_v2_0.md + #02~#06 |
| G2 | #07~#10 (T2 기술 기반) |
| G3 | #11~#16 (T2 시스템 설계) |
| G4 | #17~#22 (T3 지도/탭) |
| G5 | #23~#28 (T3 초대/프로필/온보딩) |
| G6 | #29~#34 (T3 온보딩 플로우 + T4 운영) |

→ 각 에이전트 출력: **문서명 + 불일치 항목 목록 (위치 + 현재값 + 기준값)**

### Phase 2: Fix (불일치 항목만 타겟 수정)

Phase 1 결과를 통합 → 불일치가 있는 문서만 Fix 에이전트 투입
- Edit 도구만 사용 (전체 재작성 금지)
- 불일치 없는 문서는 스킵
- 수정 후 변경 이력 섹션에 항목 추가

# SafeTrip Design System

> **버전:** v1.0 | **최초 작성:** 2026-03-03
>
> 이 문서는 SafeTrip 앱의 디자인 시스템을 정의한다.
> 모든 UI 구현은 이 문서의 토큰을 사용해야 한다.

## 5. 컴포넌트 규칙

### 5.1 버튼

- **Primary:** `primaryTeal` 배경, 흰 텍스트, 높이 52px, radius 12px
- **Secondary:** 흰 배경 + `primaryTeal` 테두리, teal 텍스트
- **Destructive:** `semanticError` 배경, 흰 텍스트
- **SOS:** `sosDanger` (#D32F2F) 배경, 흰 텍스트, 56×56dp, 2초 롱프레스

### 5.2 바텀시트 (비즈니스 원칙 §06 + 바텀시트 동작 규칙)

| Step | 이름 | 화면 비율 | 지도 노출 |
|:----:|------|:--------:|:--------:|
| 1 | `peek` | 6% | 94% |
| 2 | `collapsed` | 18% | 82% |
| 3 | `half` | 35% | 65% |
| 4 | `tall` | 55% | 45% |
| 5 | `expanded` | 75% | 25% |

- 핸들: 가운데 정렬, 높이 4px, 너비 44px, `line04` 색상
- 상단 모서리: radius 20px
- 탭 전환 시 높이/스크롤 위치 유지
- SOS 발동 시: 강제 `collapsed` 상태

### 5.3 카드

- 배경: `surface` (#FFFFFF)
- 모서리: radius 16px
- 그림자: `shadow01` (black 4% opacity)
- 내부 패딩: 16px
- 카드 간 간격: 12px

---

## 6. SOS UI 규칙 (비즈니스 원칙 §05.1)

- SOS 버튼: 화면 우하단, 바텀시트 위에 항상 노출
- 색상: `#D32F2F` (다크모드에서도 동일)
- 크기: 56×56dp, 텍스트 "SOS" 흰색 Bold
- 활성화: 2초 롱프레스 (오작동 방지)
- 활성 SOS: 전체 화면 오버레이, 빨간 배경
- SOS는 모든 프라이버시 설정을 무시하고 현재 위치 강제 포함
- 대상: 그룹 전체 멤버 + 연결된 가디언
- 접근: 캡틴/크루장/크루만 (가디언 불가)

## 7. 역할별 UI 가시성

### 7.1 메인 화면 상단 오버레이

| 요소 | 캡틴 | 크루장 | 크루 | 가디언 |
|------|:----:|:-----:|:----:|:-----:|
| 여행 정보 카드 | ✅ | ✅ | ✅ | ✅ |
| 프라이버시 등급 아이콘 | ✅ | ✅ | ❌ | ❌ |
| 위치공유 모드 표시 | ✅ | ✅ | ❌ | ❌ |
| SOS 버튼 | ✅ | ✅ | ✅ | ❌ |
| 설정/알림 아이콘 | ✅ | ✅ | ✅ | ✅ |

### 7.2 바텀시트 탭

| 탭 | 캡틴 | 크루장 | 크루 | 가디언 |
|----|:----:|:-----:|:----:|:-----:|
| 일정 | 편집 | 편집 | 읽기 | 읽기 |
| 멤버 | 관리 | 제한 관리 | 읽기 | 연결된 멤버만 |
| 채팅 | 전송 | 전송 | 전송 | ❌ |
| 안전가이드 | ✅ | ✅ | ✅ | ✅ |

### 7.3 가디언 전용 탭

- "내 멤버" 탭: 연결된 멤버 목록, 위치공유 상태, 긴급 알림
- 일정 탭: 연결된 멤버 일정 읽기 전용
- 안전가이드: 국가 안전 정보, 긴급 연락처

> **버전:** v1.0 | **작성일:** 2026-03-03
>
> 이 문서는 SafeTrip 앱의 여행 관리 플로우 16개 화면을 정의한다.

## User Journey Flow

### 일정 도메인

```
D-01 일정 탭 (바텀시트 탭 컨텐츠)
 ├── [날짜 선택] → 해당 날짜 일정 스크롤
 ├── [FAB "+"] → D-02 일정 추가 모달
 │    └── [장소 검색] → D-08 장소 검색 → D-06 장소 추가
 ├── [일정 카드 탭] → D-05 일정 상세
 │    ├── [편집] → D-02 (수정 모드)
 │    └── [삭제] → Dialog_Confirm
 ├── [빠른 입력] → D-03 빠른 일정 입력
 └── [텍스트 변환] → D-04 일정 텍스트 변환 (P1)
```

### 장소 도메인

```
D-06 장소 추가 (지도 핀)
 ├── [검색] → D-08 장소 검색
 └── [직접 입력] → D-07 장소 직접 입력

D-08 장소 검색
 └── [결과 선택] → D-06 장소 추가 (좌표 자동 입력)
```

### 멤버 도메인

```
D-09 멤버 탭 (바텀시트 탭 컨텐츠)
 ├── [멤버 카드 탭] → D-12 멤버 상세
 │    ├── [역할 변경] → Dropdown 선택
 │    ├── [멤버 내보내기] → Dialog_Confirm
 │    └── [리더 이관] → D-13 리더 이관 모달
 ├── [+ 멤버 초대] → D-10 멤버 초대
 └── [초대코드 관리] → D-11 초대코드 관리
```

### 여행 상태 도메인

```
D-14 여행 시작 체크리스트 (planning → active)
 └── [여행 시작] → C-01 메인맵 (active 상태)

D-15 여행 완료 요약 (active → completed)
 └── [여행 완료] → C-01 메인맵 (completed 상태)

D-16 여행 재활성화 (completed → active, P1)
 └── [재활성화] → C-01 메인맵 (active 상태)
```

### D-01 일정 탭 (Schedule Tab)

**메타데이터**

| 항목 | 값 |
|------|-----|
| ID | D-01 |
| 화면명 | 일정 탭 (Schedule Tab) |
| Phase | P0 |
| 역할 | 전체 (캡틴/크루장: 편집, 크루: 읽기전용, 가디언: 간접 열람) |
| 화면 유형 | 바텀시트 탭 컨텐츠 (독립 화면 아님) |
| 진입 경로 | C-01 메인맵 바텀시트 → 일정 탭 선택 |
| 이탈 경로 | D-01 → D-02 (일정 추가) / D-01 → D-05 (일정 상세) / D-01 → D-03 (빠른 입력) |

**레이아웃**

```
┌─────────────────────────────┐
│     [Layer 1] 지도 뷰        │ ← 상단 (바텀시트 위)
├─────────────────────────────┤
│  ── 핸들 바 ──               │ 4px x 44px, line04
│  [일정] [멤버] [채팅] [가이드] │ NavBar_Crew 탭 바
├─────────────────────────────┤
│                             │
│  [위치 연동 배너]             │ 프라이버시 우선 등급만 표시
│                             │
│  [ 3/15 ] [●3/16] [ 3/17 ]  │ 날짜 타임라인 바
│  [ 3/18 ] [ 3/19 ] ...      │ 가로 스크롤, 최대 15일
│                             │
│  [공유 타임라인 바]           │ 프라이버시 우선 등급만 표시
│  [00──09:00████18:00──24]   │ 녹색/회색 구간
│                             │
│  ┌─────────────────────────┐│
│  │ 09:00 │ ✈️ 인천공항 출발  ││ ListTile_Schedule
│  │       │ 인천국제공항      ││ current (좌측 teal 보더)
│  └─────────────────────────┘│
│  ┌─────────────────────────┐│
│  │ 12:00 │ 🍽️ 점심 식사     ││ ListTile_Schedule
│  │       │ 나리타 공항 식당   ││ upcoming
│  └─────────────────────────┘│
│  ┌─────────────────────────┐│
```

> 바텀시트 5단계 스냅: 일정 탭 기본 높이 `half` (35%).
> SOS 발동 시 강제 `collapsed` 전환.

**컴포넌트 명세**

| 컴포넌트 | Flutter 위젯 | 속성 |
|----------|-------------|------|
| 바텀시트 | `DraggableScrollableSheet` | style: BottomSheet_Snap, 기본 snap: half (35%) |
| 탭 바 | `TabBar` | 4탭: 일정/멤버/채팅/안전가이드, activeColor: primaryTeal |
| 위치 연동 배너 | `Container` | 배경 surfaceVariant (#F9F9F9), 텍스트 "일정 연동 공유 모드 활성", visibility: privacy_first만 |
| 날짜 타임라인 | `ListView` (horizontal) | 최대 15일, 현재 날짜 파란 테두리, 일정 있는 날짜 하단 파란 점 |
| 날짜 칩 | `Container` | width: 56dp, height: 44dp, radius8, 현재: primaryTeal 보더 |
| 공유 타임라인 바 | `CustomPaint` | 24시간 바, 녹색(공유)/연한녹색(버퍼)/회색(비공유), visibility: privacy_first만 |
| 일정 카드 | `ListTile` | style: ListTile_Schedule (높이 64dp), leading: 시간, title: 일정명, subtitle: 장소 |
| FAB 추가 버튼 | `FloatingActionButton` | icon: Icons.add, backgroundColor: primaryTeal, 크루/가디언에게 숨김 |
| 빈 상태 안내 | `Column` | "이 날짜에 등록된 일정이 없습니다" + [+ 일정 추가] 버튼 (캡틴/크루장) |

**상태 분기**

| 상태 | UI 변화 |
|------|---------|
| planning 상태 | 일정 편집 가능 (캡틴/크루장), FAB 표시, D-N 남은일 표시 |
| active 상태 | 진행 중 일정 파란 테두리 + 상단 고정, D+N 경과일 표시 |
| completed 상태 | 전체 읽기전용, FAB 숨김, 종료된 일정 회색 처리 |
| 일정 0개 | 빈 상태 안내 + [+ 일정 추가] 버튼 (캡틴/크루장) |
| 진행 중 일정 | 파란 테두리 (2px) + 상단 고정 (active 상태에서만) |
| 곧 시작 일정 | 노란 테두리 + "곧 시작" 뱃지 (버퍼 15분 이내) |
| 종료된 일정 | 텍스트/아이콘 모두 회색 (opacity 0.5) |
| 시간 충돌 | 겹치는 일정 날짜에 주황색 경고 아이콘 |
| 오프라인 | 상단 오렌지 배너 "오프라인 -- 변경 사항은 연결 후 저장됩니다" |
| 프라이버시 우선 등급 | 영역 A (위치 연동 배너) + 영역 C (공유 타임라인 바) 표시 |
| 안전최우선/표준 등급 | 영역 A, C 미표시 |

**인터랙션**

- [탭] 날짜 칩 → 해당 날짜 일정 카드로 스크롤 이동
- [스와이프 좌/우] 날짜 타임라인 → 가로 스크롤
- [탭] 일정 카드 → Navigator.push → D-05 일정 상세
- [탭] FAB "+" → showModalBottomSheet → D-02 일정 추가 모달
- [롱프레스] 일정 카드 → 컨텍스트 메뉴 (수정/삭제, 캡틴/크루장만)
- [탭] 공유 타임라인 바 특정 시간대 → 해당 시간 일정 카드로 스크롤
- [탭] 위치 연동 배너 → Navigator.push → E-01 위치 공유 설정

| 상태 | UI 변화 |
|------|---------|

| 저장 중 | 버튼 → CircularProgressIndicator |
| 저장 성공 | 모달 닫기 + D-01 일정 목록 갱신 + Toast "일정이 추가되었습니다" |
| 저장 실패 | SnackBar "일정 저장에 실패했습니다. 다시 시도해주세요." |

**인터랙션**

- [탭] 제목 필드 → 키보드 표시
- [탭] 날짜 필드 → showDatePicker (여행 기간 범위 제한)
- [탭] 시작/종료 시간 → showTimePicker
- [탭] 종일 스위치 → 시간 피커 토글
- [탭] 유형 칩 → 단일 선택 (이전 선택 해제)
- [탭] 장소 필드 → Navigator.push → D-08 장소 검색 (결과 콜백)

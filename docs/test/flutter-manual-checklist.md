# SafeTrip Flutter 수동 체크리스트

**작성일**: 2026-02-27
**환경**: Android/iOS 에뮬레이터 + Firebase Emulator
**대상 기기**: Android Emulator (API 33+), iOS Simulator (17+)

---

## 사전 설정

### 앱 빌드 및 실행
```bash
# Android 에뮬레이터
cd safetrip-mobile
flutter run -d emulator-5554

# iOS 시뮬레이터
flutter run -d booted
```

### Firebase Emulator URL 설정
- `.env` 파일에 `USE_FIREBASE_EMULATOR=true` 확인
- Android: `FIREBASE_EMULATOR_HOST=10.0.2.2`
- iOS: `FIREBASE_EMULATOR_HOST=127.0.0.1`

### 서버 실행 확인
```bash
curl http://localhost:3001/health
# 또는 Android 에뮬레이터에서: curl http://10.0.2.2:3001/health
```

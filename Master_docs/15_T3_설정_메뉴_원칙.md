## 2. 목적 및 적용 범위

### 2.1 목적

SafeTrip 설정 메뉴의 **3계층 설정 모델** (Layer 1 앱 전역 / Layer 2 여행별 / Layer 3 역할별)을 비즈니스 원칙 v5.1 기준으로 정의한다. 각 계층별 설정 항목, 접근 경로, 역할별 권한, 프라이버시 등급 연동, 가디언 과금 UI를 명세한다.

### 2.2 적용 범위

| 범위 | 포함 여부 |
|------|:--------:|
| 앱 전역 설정 (프로필, 계정, 기기 권한, 알림, 개인정보) | ✅ |
| 여행별 설정 (프라이버시 등급, 위치 공유 모드, 가디언 관리) | ✅ |

## 3. 3계층 설정 모델

### 3.1 계층 개요

```
┌────────────────────────────────────────────────────┐
│  Layer 1: 앱 전역 설정 (사용자 계정 단위)             │
│  → 프로필, 계정, 기기 권한, 전역 알림, 개인정보 관리  │
├────────────────────────────────────────────────────┤
│  Layer 2: 여행별 설정 (여행 단위)                    │
│  → 프라이버시 등급, 위치 공유 모드, 가디언 관리,      │
│     여행별 알림, 멤버 관리                           │
├────────────────────────────────────────────────────┤
│  Layer 3: 역할별 접근 권한 (같은 항목, 다른 UI)       │
│  → 캡틴: 모든 Layer 2 편집 가능                     │
│     크루장: 일부 Layer 2 읽기/편집                   │
│     크루: Layer 2 개인 설정만 편집                   │
│     가디언: Layer 2 읽기 전용 (제한적)               │
└────────────────────────────────────────────────────┘
```

### 3.2 설정 진입 경로

```
[여행 없음 상태]
  앱 우상단 프로필 아이콘 → Layer 1 설정 진입
  └─ 앱 전역 설정만 표시

[여행 참여 상태]
  앱 우상단 프로필 아이콘 → 설정 허브 진입
  ├─ 앱 설정 (Layer 1)
  ├─ [현재 여행명] 설정 (Layer 2)
  │   └─ 복수 여행 참여 시 여행 선택 드롭다운
  └─ 역할별 접근 권한 자동 적용 (Layer 3)
```

## 4. Layer 1: 앱 전역 설정

### 4.1 설정 항목 구조

```
설정
├── 프로필 관리
│   ├── 프로필 사진 변경
│   ├── 표시 이름 편집
│   ├── 전화번호 확인 (변경 불가, 인증 기반)
│   └── 언어 설정 (시스템 따름 / 수동 선택)
│
├── 계정
│   ├── 로그인 방식 확인
│   ├── 연결된 기기 목록
│   ├── 계정 로그아웃
│   └── 계정 삭제 (→ §14.4, 7일 유예)
│
├── 기기 권한
│   ├── 위치 권한: 항상 허용 / 앱 사용 중만 / 거부
│   ├── 알림 권한: 시스템 설정으로 이동
│   ├── 카메라 권한 (프로필 사진용)
│   └── 권한 현황 배지 표시 (미허용 시 경고 ⚠️)
│
├── 전역 알림
│   ├── SOS 알림: 항상 ON (비활성화 불가)
│   ├── 가디언 알림: ON/OFF
│   ├── 채팅 알림: ON/OFF
│   ├── 일정 알림: ON/OFF
│   └── 마케팅 알림: ON/OFF (초기값: 동의 여부 따름)
│
└── 개인정보 관리
    ├── 약관·동의 현황 조회
    ├── 내 정보 열람 요청 (48시간 내 이메일 발송)
    ├── 데이터 삭제 요청 (→ 계정 삭제 흐름 연결)
    └── 마케팅 동의 변경
```

### 4.2 위치 권한 처리 규칙

| 권한 상태 | 앱 동작 |
|----------|---------|
| 항상 허용 | 전체 기능 정상 동작 |
| 앱 사용 중만 | 백그라운드 위치 수집 불가 → 지오펜스 기능 제한 |
| 거부 | 위치 관련 기능 전체 비활성화 + 배너 안내 상시 표시 |

> 권한 거부 시 배너: "위치 권한이 필요합니다. 설정에서 '항상 허용'으로 변경해 주세요." (탭 시 기기 설정 이동)

## 5. Layer 2: 여행별 설정

### 5.1 설정 항목 구조 (캡틴 기준 전체 표시)

```
[여행명] 설정
├── 여행 정보
│   ├── 여행 기간: YYYY.MM.DD ~ YYYY.MM.DD (최대 15일 §02.3)
│   │   └── 수정 버튼 (캡틴 전용) → 15일 초과 불가 UI
│   ├── 여행 목적지
│   └── 여행 유형 (B2C / B2B)
│
├── 프라이버시 등급
│   ├── 현재 등급 표시: 🛡️ 안전 최우선 / 📍 표준 / 🔒 프라이버시 우선
│   ├── 변경 버튼 (캡틴 전용)
│   └── 변경 시 연쇄 영향 확인 다이얼로그 (§5.3)
│
├── 위치 공유 설정
│   ├── 내 위치 공유: ON/OFF 토글 (크루 이상 개별 제어)
│   ├── 공개 범위 (표준/프라이버시 우선 등급만)
│   │   ├── 전체 공개
│   │   ├── 캡틴·크루장에게만
│   │   └── 특정 멤버 지정
│   └── 위치 공유 일시 중지 (→ §5.4)
│
├── 가디언 관리 (★ v5.1 핵심)
│   ├── 내 가디언 목록 (→ §6 상세)
│   ├── 가디언 추가 (무료 슬롯 / 유료 슬롯 구분 표시)
│   └── 전체 가디언 목록 (캡틴 전용 조회)
│
├── 여행별 알림
│   ├── 멤버 위치 이탈 알림: ON/OFF
│   ├── 채팅 알림 (이 여행): ON/OFF
│   └── 출석 체크 알림: ON/OFF (6인 이상 유료)
│
└── 여행 관리 (캡틴 전용)
    ├── 멤버 권한 변경
    ├── 멤버 강제 탈퇴
    ├── 여행 종료
    └── 여행 삭제
```

### 5.2 여행 기간 표시 및 15일 제한 (§02.3)

**설정 화면 내 여행 기간 편집 UI:**
```
여행 기간
2026. 03. 15  →  2026. 03. 29     [편집]
             (15일, 최대)

편집 화면에서 15일 초과 시도 시:
┌─────────────────────────────────┐
│  ⚠️ 여행 기간 초과               │
│                                 │
│  SafeTrip 여행은 최대 15일까지   │
│  설정할 수 있습니다.             │
│                                 │
│  15일을 초과하는 여행은          │
│  분할하여 생성해 주세요.          │
│                                 │
│  [확인]                         │
└─────────────────────────────────┘
```

- 날짜 피커에서 종료일 선택 시 시작일 + 14일 초과 선택 비활성화 (grayed out)
- 기존 여행 기간 편집 시에도 동일 규칙 적용
- 캘린더 UI: 15일 범위 초과 영역에 빗금 처리

### 5.3 프라이버시 등급 변경 연쇄 영향

등급 변경 시 반드시 확인 다이얼로그를 표시한다:

**등급 하향 시 (안전 최우선 → 표준 / 표준 → 프라이버시 우선):**
```
┌─────────────────────────────────────────┐
│  프라이버시 등급 변경                    │
│                                         │
│  🛡️ 안전 최우선 → 📍 표준              │
│                                         │
│  변경 사항:                             │
│  • 가디언 위치 공유: 항상 → 스케줄 연동  │
│  • 가디언 일시 중지: 불가 → 2시간 허용   │
│  • 자동 위치 응답: 유지                  │
│                                         │
│  ※ 미성년자 멤버가 포함된 여행은          │
│     안전 최우선 등급만 허용됩니다.        │
│                                         │
│  [취소]              [변경 확인]         │
└─────────────────────────────────────────┘
```

**미성년자 포함 여행 등급 변경 시도 시:**
- 등급 변경 버튼 자체를 비활성화 (disabled)
- 버튼 하단 안내: "미성년자 포함 여행은 안전 최우선 등급으로 고정됩니다"

### 5.4 위치 공유 일시 중지 규칙

| 역할/등급 | 일시 중지 허용 | 최대 시간 |

  ┌─────────────────────────────┐

### 6.2 가디언 슬롯 상태별 UI

| 상태 | 무료 슬롯 표시 | 유료 슬롯 표시 |
|------|-------------|-------------|
| 무료 슬롯 0/2 사용 | "무료 슬롯 (0/2)" + [추가] 활성 | 숨김 |
| 무료 슬롯 1/2 사용 | "무료 슬롯 (1/2)" + [추가] 활성 | 숨김 |
| 무료 슬롯 2/2 사용 | "무료 슬롯 (2/2)" | 유료 슬롯 섹션 표시 |
| 무료 2 + 유료 1 이상 | 무료 만료 표시 | 유료 슬롯 (N/3) 표시 |

### 6.3 추가 과금 모달 (3번째 가디언 추가 시)

무료 슬롯 2개가 모두 사용된 상태에서 가디언 추가 시도 시 표시:

```
┌────────────────────────────────────────┐
│           가디언 추가 과금 안내          │
│                                        │
│  무료 가디언 2명을 모두 사용했습니다.    │
│                                        │
│  ┌──────────────────────────────────┐  │
│  │  추가 가디언 1명                  │  │
│  │  1,900원 / 여행                  │  │
│  │  (이 여행에만 적용됩니다)          │  │
│  └──────────────────────────────────┘  │
│                                        │
│  • 최대 3명까지 유료 추가 가능          │
│  • 해제 시 환불되지 않습니다           │
│  • 안전 기능은 무료/유료 무관          │
│    동일하게 제공됩니다               │
│                                        │
│  [취소]           [결제 후 추가]       │
└────────────────────────────────────────┘
```

결제 후 추가 버튼 탭 시:
1. 앱 스토어(In-App Purchase) 결제 화면 표시
2. 결제 완료 → `tb_payment` INSERT (`payment_type: 'addon_guardian'`)
3. 가디언 초대 코드 또는 전화번호 입력 화면으로 이동
4. 결제 취소 → 모달 닫기 (가디언 추가 취소)

## 7. 계정 삭제 프로세스 (§14.4)

### 7.1 삭제 요청 흐름

```
[설정 → 계정 → 계정 삭제]

Step 1. 확인 화면
  "정말 계정을 삭제하시겠습니까?"
  ├─ 삭제 후 처리 안내:
  │   - 즉시 삭제: 프로필, 이미지, 긴급 연락처
  │   - 익명화 보관: 위치 데이터, 이벤트 로그
  │   - 영구 보관: SOS 기록 (법적 의무)
  │   - 7일 유예 기간 내 철회 가능
  └─ active 여행 참여 중이면:
      "참여 중인 여행 [여행명]에서 탈퇴됩니다."
      여행 탈퇴 처리 후 계정 삭제 진행

Step 2. OTP 재인증 (보안 확인)
  전화번호로 6자리 OTP 발송 → 입력 확인

Step 3. 삭제 요청 완료
  "7일 후 계정이 삭제됩니다."
  tb_user.deletion_requested_at = NOW()

Step 4. 7일 이내 철회 가능
  로그인 후 "계정 삭제 요청 중 — [철회하기]" 배너 상시 표시
  [철회하기] → deletion_requested_at = NULL
```

### 7.2 미성년자 계정 삭제

14세~17세 미성년자: 보호자 동의 확인 후 삭제 처리 (§10.1)

## 11. 에러 및 엣지케이스 처리

| 상황 | 처리 방식 |
|------|---------|
| 가디언 무료 슬롯 초과 시도 | 과금 안내 모달 표시 (§6.3) |
| 유료 가디언 3명 모두 사용 후 추가 시도 | "가디언은 최대 5명까지 연결 가능합니다 (무료 2 + 유료 3)" 안내, 버튼 비활성화 |
| 미성년자 포함 여행 프라이버시 등급 변경 | 변경 버튼 비활성화 + 안내 문구 표시 |
| 15일 초과 여행 기간 입력 | 날짜 피커에서 초과 범위 선택 불가 + 안내 다이얼로그 |
| 미성년자 크루가 가디언 일시 중지 시도 | "미성년자는 위치 공유를 일시 중지할 수 없습니다" 안내 |
| 미성년자 크루의 가디언 해제 시도 | "캡틴 승인이 필요합니다" 안내 → 캡틴에게 승인 요청 푸시 |
| 계정 삭제 요청 중 active 여행 참여 | 여행 탈퇴 안내 → 동의 후 탈퇴 후 삭제 진행 |
| 설정 저장 중 네트워크 단절 | "변경 사항을 저장하지 못했습니다. 재시도 중…" 토스트 표시, 로컬 큐잉 |
| 결제 실패 (가디언 추가 과금) | "결제에 실패했습니다. 다시 시도해 주세요." 모달 + 재시도 버튼 |
| 유료 가디언 해제 | "해제 후 환불되지 않습니다. 해제하시겠습니까?" 확인 다이얼로그 |

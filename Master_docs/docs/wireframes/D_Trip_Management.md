## 1. 개요

- **화면 수:** 12개 (D-01 ~ D-16, D-03/D-04 제거, D-06/D-07 → D-02 하위 단계로 병합)
- **Phase:** P0 11개, P1 1개 (D-16 재활성화)
- **핵심 역할:** 캡틴 (전체 관리), 크루장 (제한 관리), 크루 (읽기전용), 가디언 (간접 열람/상태탭 전용)
- **여행 상태:** planning (일정 편집), active (실시간 추적), completed (열람 전용)
- **핵심 스펙:**
  - 일정 5영역 (A~E): A(위치 연동 배너), B(날짜 타임라인), C(공유 타임라인), D(일정 카드 목록), E(추가 버튼)
  - 일정 7아이콘: 숙소(stay), 식사(meal), 관광(sightseeing), 쇼핑(shopping), 교통(move), 기타(other), 모임(meeting)
  - 버퍼존 15분 (캡틴 설정 가능: 0/15/30분)
  - 여행 기간 최대 15일 (§02.3)
  - 멤버 4섹션 정렬: SOS 발신자 → 역할순(캡틴→크루장→크루) → 공유상태(온라인→오프라인) → 가디언(무료→유료)
  - 가디언 무료 2명 / 유료 최대 3명 분리 표시 (§03.1, §09.3)
  - 초대코드 7자리 (혼동 문자 제외, §23)
  - SOS: 3초 롱프레스 발동


#### 일정 카드 상태 표시

| 일정 상태 | 시각적 처리 | 조건 |
|----------|-----------|------|
| A — 미래 확정 | 기본 스타일 (흰색 배경) | start_datetime > 현재 시각 (버퍼 외) |
| B — 현재 진행 | 파란 테두리 2px + 상단 고정 | start_datetime <= 현재 <= end_datetime (active 여행만) |
| C — 다음 예정 | 노란 테두리 + "곧 시작" 배지 | 버퍼 15분 이내 (start_datetime - 15분 <= 현재 < start_datetime) |
| D — 완료 | 텍스트/아이콘 회색 (opacity 0.5) | end_datetime < 현재 |
| E — 취소/건너뜀 | 취소선 + 회색 | deleted_at IS NOT NULL 또는 status = 'skipped' |

#### 장소 입력 하위 단계 (D-06/D-07 병합)

장소 필드 탭 시 3가지 입력 방식을 제공한다:

**방식 1: 장소 검색 (→ D-08)**
- 탭 시 D-08 장소 검색 화면으로 이동
- 검색 결과 선택 시 장소명+좌표 콜백

**방식 2: 지도 핀 선택 (구 D-06)**
- 인터랙티브 지도에서 핀 위치를 드래그하여 장소 지정
- 중앙 고정 핀 + 역지오코딩 텍스트 표시
- 지도 60% 높이 + 하단 장소명/카테고리/메모 입력

**방식 3: 직접 입력 (구 D-07)**
- 주소 탭: 도로명/지번 주소 입력 → 지오코딩 → 미니맵 확인
- 좌표 탭: 위도/경도 직접 입력 → 역지오코딩 → 미니맵 확인

### 3.1 일정 도메인

| 상황 | 처리 |
|------|------|
| 15일 초과 날짜 선택 시도 | 날짜 선택기에서 범위 외 비활성화 + Toast "여행 기간 내의 날짜를 선택해 주세요" |

| 종료 시간 < 시작 시간 | 저장 차단 + 에러 "종료 시간이 시작 시간보다 빠릅니다" |

| 크루가 일정 수정/삭제 시도 | 버튼 비활성화 (UI 차단), API 403 → "권한이 없습니다" 토스트 |

| 일정 0개 | "이 날짜에 등록된 일정이 없습니다" + [+ 일정 추가] (캡틴/크루장) |
| 일정 삭제 확인 | Dialog_Confirm "이 일정을 삭제하시겠습니까? 삭제된 일정은 복구할 수 없습니다." |
| 진행 중 일정 수정 시도 (active) | 경고 다이얼로그 "진행 중인 일정을 수정합니다" 후 수정 진행 |

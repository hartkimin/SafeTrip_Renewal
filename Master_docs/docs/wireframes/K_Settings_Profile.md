## 1. 개요

- **화면 수:** 8개 (K-01 ~ K-08)
- **Phase:** P0 4개 (K-01, K-02, K-03, K-05), P1 4개 (K-04, K-06, K-07, K-08)
- **핵심 역할:** 캡틴/크루장/크루/가디언 (전체)
- **핵심 설계 원칙:** 3-Layer 설정 모델 (Layer 1 앱 전역 / Layer 2 여행별 / Layer 3 역할별 접근 권한)
- **주요 비즈니스 규칙:**
  - 계정 삭제 7일 유예 (§14.4)
  - 프라이버시 등급 변경 시 캐스케이드 확인 다이얼로그 필수 (§02.7)
  - 가디언 무료 2명 / 유료 3명, 3번째부터 1,900원/여행 과금 (§03.1, §09.3)
  - 위치 권한 미허용 시 경고 배너 상시 표시
  - SOS 알림 해제 불가 (§05.1)
  - 여행 기간 최대 15일 (§02.3)

### User Journey Flow

```
[C-01 메인맵 상단바 ⚙️] ──→ K-01 설정 메인
                              ├── [프로필 카드 탭] ──→ K-02 프로필 편집
                              ├── [위치 설정] ──→ K-03 위치 설정
                              ├── [알림 설정] ──→ K-04 알림 설정 (P1)
                              ├── [앱 정보] ──→ K-05 앱 정보
                              │                  └── [약관 링크] ──→ WebView
                              ├── [약관 재동의] ──→ K-06 약관 재동의 (P1, 업데이트 시)
                              └── [계정 관리]
                                   ├── [로그아웃] ──→ Dialog_Confirm ──→ A-01
                                   ├── [계정 삭제] ──→ K-07 계정 삭제 (P1)
                                   │                  └── [삭제 요청 완료] ──→ A-01
                                   └── [삭제 철회] ──→ K-08 삭제 철회 (P1, 유예 기간 중)
                                                      └── [철회 완료] ──→ K-01
```

### 3-Layer 설정 모델 개요

```
┌────────────────────────────────────────────────────┐
│  Layer 1: 앱 전역 설정 (사용자 계정 단위)             │
│  → 프로필, 계정, 기기 권한, 전역 알림, 개인정보 관리  │
├────────────────────────────────────────────────────┤
│  Layer 2: 여행별 설정 (여행 단위)                    │
│  → 프라이버시 등급, 위치 공유 모드, 가디언 관리,      │
│     여행별 알림, 멤버 관리                           │
├────────────────────────────────────────────────────┤
│  Layer 3: 역할별 접근 권한 (같은 항목, 다른 UI)       │
│  → 캡틴: 모든 Layer 2 편집 가능                     │
│     크루장: 일부 Layer 2 읽기/편집                   │
│     크루: Layer 2 개인 설정만 편집                   │
│     가디언: Layer 2 읽기 전용 (제한적)               │
└────────────────────────────────────────────────────┘
```

## 3. 에러 & 엣지케이스

| # | 케이스 | 처리 방법 |
|---|--------|----------|
| 1 | **위치 권한 거부 상태에서 여행 참여** | K-01/K-03에 경고 배너 상시 표시: "위치 권한이 필요합니다. 설정에서 '항상 허용'으로 변경해 주세요." 탭 시 시스템 설정 이동 |
| 2 | **가디언 무료 슬롯 초과 시도** | 과금 안내 모달 표시 (§09.3): 1,900원/여행/명, 최대 3명 유료 추가 가능 |
| 3 | **유료 가디언 5명 모두 사용 후 추가 시도** | "가디언은 최대 5명까지 연결 가능합니다 (무료 2 + 유료 3)" 안내, 버튼 비활성화 |
| 4 | **미성년자 포함 여행 프라이버시 등급 변경 시도** | 변경 버튼 비활성화 + "미성년자 포함 여행은 안전 최우선 등급으로 고정됩니다" 안내 |
| 5 | **15일 초과 여행 기간 입력** | 날짜 피커에서 시작일+14일 초과 선택 비활성화 + 안내 다이얼로그: "SafeTrip 여행은 최대 15일까지 설정할 수 있습니다" |
| 6 | **미성년자 크루가 위치 일시 중지 시도** | "미성년자는 위치 공유를 일시 중지할 수 없습니다" 안내, 토글 비활성 |

| 10 | **설정 저장 중 네트워크 단절** | Toast "변경 사항을 저장하지 못했습니다. 재시도 중..." + 로컬 큐잉 |

| 가디언 추가/해제 | ❌ 온라인 필수 (결제 및 초대 코드 발행 필요) |

| 약관 동의 | ❌ 온라인 필수 |
| 앱 정보 조회 | ✅ 버전 정보 로컬 캐시, 약관 전문은 ❌ (WebView 필요) |

> 오프라인 상태에서 온라인 필수 기능 접근 시: "이 기능은 인터넷 연결이 필요합니다. 연결 후 다시 시도해 주세요." 안내 표시.
